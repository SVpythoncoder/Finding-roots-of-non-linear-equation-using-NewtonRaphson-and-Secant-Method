Here is a simple raw program which help to implement the Simpson 1/3rd Rule.

DETAILS:-

Input required from user:-
One needs to write the function in the code itself

A user interface for providing – 
(i) interval extreme points i.e 'a' and 'b' 
(ii) The value of n 

Output generated:-
(i) Result of the Simpson 1/3rd method
