# Finding the solution using Thomas Algorithm

Here is a simple raw program which help to implement the Thomas Algorithm for finding the solution matrix (X matrix)

DETAILS:-

Inputs required from user:-
A user have to input the following data in the code-
(i) Lower diagonal of the Tridiagonal Matrix
(ii) Mains diagonal of the Tridiagonal Matrix
(iii) Upper diagonal of the Tridiagonal Matrix

Output generated:-
(i) The Unknown matrix (X matrix)
